# 🌐 Language Translation Tool
### CodeAlpha AI Internship — Task 1

A web-based Language Translation Tool built using Python, Flask, and Google Translate API.

---

## 🚀 Features
- 🌍 Supports 15+ languages (Hindi, English, French, Arabic, Urdu, Japanese, etc.)
- 🔍 Auto language detection
- 📋 Copy translated text with one click
- 🔊 Text-to-Speech for translated output
- 🗑️ Clear button to reset input/output
- 🔢 Character counter
- 💻 Clean dark glassmorphism UI

---

## 🛠️ Tech Stack
- **Backend**: Python, Flask
- **Translation**: Google Translate API (`googletrans==4.0.0rc1`)
- **Frontend**: HTML, CSS, JavaScript

---

## ⚙️ How to Run

**1. Clone the repository**
```bash
git clone https://github.com/kushkk28khandelwal/CodeAlpha_LanguageTranslation.git
cd CodeAlpha_LanguageTranslation
```

**2. Install dependencies**
```bash
python3 -m pip install -r requirements.txt
```

**3. Run the app**
```bash
python3 app.py
```

**4. Open in browser**
```
http://127.0.0.1:5001
```
